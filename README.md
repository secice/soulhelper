# SOUL App 消息防止撤回插件

## 使用

### Xposed

支持Xposed，直接安装，然后在Xposed Installer里面启用插件即可

### VirtualXposed

支持VirtualXposed, 在VirtualXposed里面添加soul，然后添加插件，启用插件即可
